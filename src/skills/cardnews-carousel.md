## Card News Carousel — Direction Guide

구현 정확성보다 먼저 **"고밀도, 에디토리얼, 완성도 높은 카드뉴스 디자인"**을 우선한다.

---

## 1. 비주얼 목표 (MOST IMPORTANT — 이것부터 읽어라)

- 레퍼런스 느낌: 스타트업 인사이트 카드뉴스 + 프리미엄 SNS 에디토리얼
- **화면이 듬성듬성 비어 보이면 실패**
- 모든 씬은 4:5 캔버스에서 시각적으로 **82~90% 이상을 사용**
- 텍스트를 그냥 놓지 말고, 반드시 **패널/카드/장식 요소와 함께** 배치
- 중앙에 작은 요소를 띄우는 구도 금지
- 각 씬은 명확한 **시각적 덩어리 2~4개**로 구성
- 단순한 PPT처럼 보이지 말고, **카드뉴스 디자이너가 만든 것처럼** 보여야 함
- 배경에는 최소 **2개의 장식 레이어** 사용: 큰 반투명 패널, 얇은 라인/그리드/워드마크 중 택2

---

## 2. 씬별 디자인 규칙

### Hero (Cover)
- 좌측 정렬
- 대제목은 화면의 시각적 중심을 크게 차지
- 배경에 초대형 반투명 워드마크 또는 기하학 장식 배치
- 하단 CTA/서브카피 영역은 독립 박스로 강조
- 빈 공간이 남으면 큰 배경 패널로 메워라

### List
- 항목 4개를 세로로 빽빽하게 배치
- 각 항목은 높이 110~120px의 큰 카드
- 좌측 16px 컬러 바, emoji 아이콘, 제목, 우측 pill 태그 필수
- 항목 사이 간격 18~22px
- 각 카드 뒤에 약한 tinted panel을 깔아 단조로움 방지

### Split
- 좌우 2개 카드가 각각 화면 너비의 42~44% 차지
- 중앙 간격은 24px 내외
- left는 경고 느낌, right는 해결 느낌
- 상단 title, 중앙 비교 카드, 하단 강조 박스까지 꽉 차게 구성

### Flow
- 3개 step 카드를 연결 구조로 배치
- 번호 원형 배지와 연결선 필수
- 단순 가로 나열보다 리듬감 있게 배치
- 각 step 카드는 충분히 크고, 전체적으로 중앙을 꽉 채워야 함

### Focus
- quote를 매우 크게 사용
- 큰 따옴표 장식, accent side bar, 반투명 배경 패널 필수
- quote만 덩그러니 두지 말고, 장식 요소로 밀도감 확보

### Final Hero (Closing)
- "BRAND"를 초대형 배경 워드마크로 사용
- CTA 박스를 전면 하단에 크게 배치
- URL은 작은 보조정보로 두되, 전체 구성은 강한 엔딩 카드처럼 설계

## 3. 타이포그래피 + 간격 + 점유율

### 타이포 규칙
| 역할 | 사이즈 | Weight | 비고 |
|------|--------|--------|------|
| Cover 타이틀 | 72~92px | 800 | line-height 0.95~1.05, 최대 3줄 |
| Body 타이틀 | 52~64px | 800 | 최대 2줄 |
| Body 텍스트 | 30~38px | 400 | 항목, 설명 |
| 항목 라벨 | 30~38px | 600 | emoji 뱃지 옆 |
| Quote | 96~120px | 800 | Focus 모드 |
| 태그/pill | 22~26px | 500 | pill badge 내부 |
| 캡션/면책 | 18~22px | 300 | 50% opacity |
| 강조 바 텍스트 | 30~38px | 600 | accent bg |

**한국어 긴 제목 규칙:**
- 의미 단위로 줄바꿈해 **시각적으로 큰 덩어리**를 유지
- **폰트를 줄여서 해결하지 말고** 줄바꿈과 블록 크기로 해결
- 한 줄 최대 12~15자 (한글 기준)

### 간격/점유율 (숫자로 강제)
- 모든 씬은 캔버스 면적의 **82~90%**를 시각적으로 사용
- 어느 사분면도 **15% 이상 비어 보이지 않게**
- 메인 콘텐츠 블록 너비: 캔버스의 **84~90%**
- 외곽 패딩: **56~64px**
- 콘텐츠 블록 간 간격: **20~28px**
- 항목 사이 간격: **8~12px**
- 텍스트만 떠 있지 말고 항상 **배경 패널/카드 위에 배치**
- 작은 요소를 화면 중앙에 띄우지 말고 **큰 블록 중심**으로 구성
- 하단 강조 박스: 모든 본문 씬에서 반드시 크게 사용, 높이 **150~180px**

## 4. 색상 팔레트

### 태정 스타일 (기본)
```
배경:        #1a1a2e (다크 차콜)
배경 카드:    #0a1628 (더 어두운 네이비)
배경 장식:    #2d2d44 (반투명 패널용)
악센트:      #00AEEF (시안 블루)
악센트 라이트: #E0F7FF
텍스트:      #ffffff
텍스트 서브:  #ffffffb3 (70% hex: B3)
보더:        #ffffff1a (10% hex: 1A)
```

### 항목별 색상 (list 모드)
```
1: #ef4444 (빨강) — 배경: #ef444426 (15% hex: 26)
2: #f97316 (주황) — 배경: #f9731626
3: #8b5cf6 (보라) — 배경: #8b5cf626
4: #ec4899 (분홍) — 배경: #ec489926
5: #22c55e (초록) — 배경: #22c55e26
```

**색상 규칙:** 8-digit hex만 사용. rgba 금지. `#RRGGBBAA` 형식.

## 5. 장식 요소 (밀도를 올리는 핵심)

모든 씬에 아래 중 **최소 2개** 사용:

- **배경 패널**: bg card(#0a1628) 위에 콘텐츠. 단일 평면 배경 위 텍스트 금지
- **오버사이즈 타이포**: 배경에 200~400px 크기 워드마크 (opacity 0.04~0.06)
- **얇은 보조 그리드**: 1px accent 라인, 60px 간격, opacity 0.06
- **숫자 배지**: 원형 accent bg, 40~48px, weight 800
- **라벨 칩/pill**: border 1px, rounded-full, 작은 텍스트
- **side bar**: 좌측 accent 세로 바 3~4px + 서브타이틀
- **corner accent**: 모서리에 작은 장식 도형 (삼각형, 원호)
- **큰 quote mark**: 인용 슬라이드에서 160~200px " 기호
- **gradient overlay**: 하단→상단 fade, 섹션 구분용

## 6. 구현 제약 (마지막에 적용)

- **`<Series>`**로 슬라이드 연결. 각 슬라이드 durationInFrames: 90
- 각 슬라이드는 **별도 함수 컴포넌트**, useCurrentFrame()으로 로컬 프레임
- spring + interpolate + clamp만 사용
- 색상은 **8-digit hex만** (#RRGGBBAA). rgba() 금지
- 한글은 반드시 **const 변수**에 넣고 JSX에서 참조
- `export const MyAnimation = () => { ... };` (React.FC 금지)
- 폰트: `Inter, system-ui, sans-serif`

### 애니메이션 시퀀싱
- snappy spring: { damping: 15, stiffness: 200, durationInFrames: 25 }
- 헤더(0f) → 배경 장식(0f) → 타이틀(5f) → 서브(10f) → 항목 순차(15f, 간격 4f) → 강조 바(50f)
- 항목: translateY(20→0) + opacity(0→1)
- 강조 바: translateY(30→0) + opacity
