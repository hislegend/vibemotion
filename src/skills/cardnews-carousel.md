## Card News Carousel — Direction Guide

인스타그램 카드뉴스(카루셀) 슬라이드를 생성한다.
`<Series>`로 슬라이드를 순차 연결. 각 슬라이드는 별도 함수 컴포넌트.

## 핵심 규칙

1. **해상도:** 1080×1350 (4:5). 다른 비율도 지원.
2. **슬라이드 최대 20장.**
3. **폰트:** Inter, system-ui, sans-serif.
4. **한글은 반드시 const 변수에 넣고 JSX에서 참조.** 템플릿 리터럴 안에 한글 직접 넣지 않는다.
5. **`<Series>`로 슬라이드 연결.** 각 슬라이드 durationInFrames: 90 (3초).
6. **콘텐츠가 화면을 꽉 채워야 한다.** 과도한 여백 금지.

## 레이아웃 밀도

- **세이프존:** 상 40px / 하 40px / 좌우 36px (최소한으로)
- **콘텐츠 면적: 70~80%.** 빈 공간은 20~30%만.
- **항목 간격: 8~12px.** 16px 초과 금지.
- **카드/바 패딩: 16~24px.** 콘텐츠가 들어찬 느낌.
- **타이틀→콘텐츠 간격: 16~20px.** 바로 이어지는 느낌.

## 타이포그래피

| 역할 | 사이즈 | Weight |
|------|--------|--------|
| 표지 헤드라인 | 48-52px | 900 (black) |
| 본문 타이틀 | 32-36px | 800 |
| 본문 서브 | 20px | 400 |
| 항목 라벨 | 18px | 600 |
| 본문 텍스트 | 20px | 400 |
| 태그/pill | 14px | 500 |
| 캡션/면책 | 14px | 300, 50% opacity |
| 강조 바 텍스트 | 20px | 600 |

## 애니메이션

- snappy spring: { damping: 15, stiffness: 200 }
- 시퀀싱: 헤더(0f) → 타이틀(5f) → 서브(10f) → 본문(15f) → 항목 순차(20f, 간격 4f) → 강조 바(50f)
- 항목: translateY(20→0) + opacity(0→1)
- 강조 바: scaleX(0→1) 또는 opacity + translateY

## Avoid

- **과도한 여백** — 콘텐츠가 70% 미만이면 허전
- 항목 간 16px 초과 간격
- 강조 박스 없는 본문 슬라이드
- 단순 번호 뱃지만 (emoji 아이콘 필수)
- 템플릿 리터럴 안에 한글 직접 넣기
- React.FC 타입 어노테이션 사용
- 좌정렬과 중앙정렬 혼용
